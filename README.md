# SpoolPainter

SpoolPainter iOS application for managing 3D printer filament spools with NFC tag integration.

## Features
- NFC tag reading and writing
- Integration with Spoolman
- Filament tracking
- Material management
- Temperature monitoring

## Building
Requires Xcode with iOS SDK support.

Many thanks and credits goes to ni4223 who made the original Android version that inspired me to make this iOS version.
It is also uploaded to the Apple Appstore, and hopefully Apple will make it visible soon.
